
# Airline Reservation System

## Description
A web-based Airline Reservation System allowing users to search flights, book tickets, choose meals, make payments, and manage bookings.

## Features
- Login & Signup
- Flight search with multiple airports
- Booking & cancellation
- Food & beverages selection
- Payment options (UPI, Card, QR)
- My Bookings section
- Contact & Support page

## How to Use
1. Open the website.
2. Register or log in.
3. Search for flights.
4. Select flight, passengers, meals.
5. Proceed to payment.
6. View or cancel bookings in “My Bookings”.

## Technologies Used
- HTML5

- JavaScript
- Java 
- Database
