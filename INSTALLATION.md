# Installation Guide

Follow these simple steps to run the Airline Reservation System locally:

## 1. Download the Project
Download the ZIP from GitHub or clone using:
```
git clone [your-repo-link]
```

## 2. Open the Project
Extract the ZIP (if downloaded).

## 3. Run the Website
Simply open:
```
index.html
```
in **any browser** (Chrome, Edge, Firefox).

## 4. Optional: Run Java Backend
Open `/java-backend/` files in any IDE (VS Code, IntelliJ, Eclipse).

You're all set!
